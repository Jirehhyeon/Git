## 📝 학습 노트

### Git 기본 개념
- **Git**: 분산 버전 관리 시스템
- **Repository**: 프로젝트의 모든 파일과 변경 이력
- **Commit**: 변경사항의 스냅샷
- **Branch**: 독립적인 개발 라인

### 주요 명령어
```bash
# 저장소 초기화
git init

# 파일 추가 (스테이징)
git add <파일명>
git add .  # 모든 파일

# 커밋 (변경사항 저장)
git commit -m "커밋 메시지"

# 원격 저장소 연결
git remote add origin <GitHub_주소>

# 원격 저장소에 업로드
git push origin main
```
